RiTa and Twitter4j external jars. To add these libraries to your Eclipse project:



- Right click the name of your Eclipse project, go to Properties

- Select "Java Build Path" from the list, then select the "Libraries" tab.

- Click the "add External JARs" button, navigate to the /src/external_libs/ folder, and add rita.jar and twitter4j-core-4.0.4.jar to the build path.

- Click apply and ok to close the dialog.
